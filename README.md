This repository contains a simple project developed in Ruby, created for study and practice purposes.

---

**<h2>Installation</h2>**

###

**Windows**

###

- Access: [https://rubyinstaller.org/downloads/](https://rubyinstaller.org/downloads/);
- Download the installer: `Ruby` + `Devkit`;
- During installation, check: *Add Ruby executables to your PATH*;
- When finished, spin `ridk install` 1st option;
- See if it's running:

###
```powershell
ruby -v

###

## Run the Application

###

Inside your project's folder, type

###
```powershell
ruby name_of_file.rb
```
